OWASP Top 10: 2021 vs 2025

The evolution of web security risks in the age of Cloud & AI.

Rank 2021 Category 2025 Updated Category Key Evolution
A01 Broken Access Control Broken Access Control Remains #1. Now explicitly includes SSRF (Server-Side Request Forgery).
A02 Cryptographic Failures Security Misconfiguration Rose from #5 to #2. Reflects the shift toward Cloud & Infrastructure-as-Code.
A03 Injection Software Supply Chain Failures NEW Rank. Expanded from "Outdated Components" to cover the entire build pipeline.
A04 Insecure Design Cryptographic Failures Moved down. Focus remains on sensitive data exposure and weak hashing.
A05 Security Misconfiguration Injection Fell from #3 to #5. Still critical, but better defended by modern frameworks.
A06 Vulnerable Components Insecure Design Focuses on design flaws rather than implementation bugs.
A07 Identification & Auth Authentication Failures Streamlined name; focus on MFA bypass and session hijacking.
A08 Software/Data Integrity Software or Data Integrity Focuses on CI/CD pipeline security and malicious updates.
A09 Logging & Monitoring Logging and Alerting Failures Updated to emphasize the need for active Alerting, not just logs.
A10 SSRF Mishandling of Exceptional Conditions NEW Category. Focuses on logic errors and unhandled crashes.
